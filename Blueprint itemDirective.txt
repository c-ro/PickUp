Item directive locations:
1. add-to-list-dialog.html
2. items.html
3. list.html

Item directive functions (/directives/item.js):
  -react when clicked:
    -lowlight/put in cart (3)
    -unLowlight/take out of cart(3)
    -highlight/add to current list(1)
    -unHighlight/remove from current list(1)
  -add to master list
  -remove from master list
  -edit all item properties from master list
  -edit price and qty from current list
  -offer to send user to master list if they try to change [item.name]

Item directive elements (itemDirective.html):
  -crossout button
  -item name
  -item qty
  -item unitprice
  -item subtotal
  -add item to current list 
  -remove item from current list
  -add item to main list
  -delete item from main list


List Contexts:
  Master List
    -[CONTENTS]
      - displays all item ever added to list
      -displays name per item
      -displays most recent qty-price per item
      -displays avg/total qty/price per item
      -displays link to item history per item
    
    -[ACTIONS]
      -view item history
      -remove item from master
      -add item to current (default, or user selection)


  Intermediate List
    -[CONTENTS] All items not in current list
      -displays name per item
      -most recent price per item
      -last purchase date per item

    -[ACTIONS]
      -add item to current 
      -remove item from current list (before dialog close)
      -view purchase history

  Current List
    -[CONTENTS] Items generated by app, not unselected by useritems or explicity selected by user
      -displays name per item
      -displays most recent subtotal price per item

    -[ACTIONS]
      - crossout item
      - un-crossout item
      - remove from current list
      - view/edit price details
